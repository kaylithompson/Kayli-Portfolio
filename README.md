# [Project 1- Kaggle Home Credit Risk Default](https://github.com/kaylithompson/home-credit-project)

## Project Overview:
The goal of this project is to build a model that predicts the likelihood a client will default on a loan using features from the Home Credit dataset.

## [Preprocessing Functions](https://github.com/kaylithompson/home-credit-project/blob/main/credit_risk_preprocessing.R)
Created a set of reusable functions for cleaning, transforming, and engineering features from credit risk application data, ensuring consistent processing for both training and test sets.

## [LightGBM Model](https://github.com/kaylithompson/home-credit-project/blob/main/Home%20Credit%20Risk%20Model.R)
The LightGBM model predicts loan default risk using applicant features, enabling accurate, data-driven lending decisions.

## [Modeling Card](https://github.com/kaylithompson/home-credit-project/blob/main/Modeling%20Card.qmd)
This modeling card documents the development, evaluation, and business impact of a LightGBM-based credit risk model for Home Credit. It summarizes the model’s purpose, performance, decision logic, explainability, fairness analysis, and key limitations to support transparent, responsible deployment.
